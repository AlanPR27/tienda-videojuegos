# Tienda de Videojuegos - Proyecto Final

## Descripción
Este proyecto consiste en el desarrollo de una tienda en línea de videojuegos, consolas y accesorios utilizando tecnologías web como HTML, PHP, MySQL y Bootstrap.

El sistema permitirá a los usuarios registrarse, iniciar sesión, explorar un catálogo de productos, agregar artículos a un carrito de compras y realizar compras.

---

## Funcionalidad actual (avance)
- Navegación entre páginas
- Diseño base con Bootstrap
- Estructura del sistema:
  - Inicio
  - Catálogo
  - Login
  - Registro
  - Carrito
  - Panel de administración
